Tables

  • symbol:
    • table element:<table> & </table>
    • table row:<tr> & </tr>
    • table cell:<td> & </td>
    • example:
      <table>
         <tr>
           <td>
               [cell element]
           </td>
         </tr>
      </table>
  • border:Two ways to add boarder in the table
    1. CSS: better solution than HTML
    2. HTML: <table border="1"> => old way, not recommand
  • Header:
    • symbol: <th> & </th>
    • example:
      <table border="1">
         <tr>
           <th>
               [Header cell]
           </th>
         </tr>
         <tr>
           <td>
               [cell element]
           </td>

Forms

Getting User Input, a container of some of inputs
  • tags: <form>,<input>,<label>
  • Form tag:
    • create an HTML form for user input, contains more of the form elements
    • format:
      <form action="[URL]" method="[method(post/get)]">
      [All inputs in here]
      </form>
      • action: the URL to send form data to
      • method: the type of HTTP request
      • * forms won't actually do anything, unless it connects with the backend JS. Currently, just frontend
  • Input tag:
    • The input tag creates interactive controls.
    • type attribute:
      1. text:
      2. date:
      3. color:
      4. file:
      5. checkbox:
      6. radio:
        • example for only one radio can be choose:
          name attribute: identical name can force only one radio to be choose
          value attribute: the value will be showed in the url
      7. password:
      8. sumbit:
        substitute: <button>Submit Sub</button>
      9. checkbox:
    • placeholder attribute:<input type="[type]" placeholder="[text]">
    • name attribute: give name of data and trace later => backend
  • Select tag: a dropdown menu
    • Option tag: put the selected item
    • example:

      <select name="[name of the select (color...)]">
          <option value="[value of the option]">
             [item]
         </option>
      </select>
  • textarea tag:
    • Format:
      <textarea name="[name of textarea]" id="" cols="30" rows="10"><textarea>
